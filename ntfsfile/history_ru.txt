Версия: 2.0.0 19.12.2006
- поддержка Windows Vista
- более удобный просмотр длинных путей (нажатие Enter или двойной щелчок по полю)
Функции анализа данных для выбранного файла:
- оценка сжимаемости данных файла (производится сжатие данных с помощью быстрого алгоритма LZO, полезно чтобы определить - стоит ли сжимать файл средствами NTFS или архиватором)
- подсчёт объёма кластеров, целиком заполненных нулями (полезно чтобы определить - стоит ли делать файл разрежённым)
- расчёт наиболее полезных хешей: crc32, md5, sha1, ed2k (вариант eMule).
